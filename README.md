# Marginal

A Chrome extension for saving research quotes with source citations.

## Setup

1. Clone the repo
2. Generate the extension icons:
   ```
   python3 generate-icons.py
   ```
3. Open Chrome → `chrome://extensions`
4. Enable **Developer mode**
5. Click **Load unpacked** and select this folder

## Usage

- Highlight text on any webpage
- Press `⌘⇧S` (Mac) or `Ctrl+Shift+S` to save the quote
- Click the Marginal icon to open the side panel
- Use the **Export** tab to copy all quotes in APA, MLA, or Chicago format
